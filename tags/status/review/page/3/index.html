EHR Core Values

September 17, 2021

EHR The original goals behind EHRs were to make claims flow more efficiently and I would argue that in many ways they are doing that. Our goal when set out to build Dorsata was to build the missing specialty specific clinical layer and top of the existing EHR. While leveraging the robust ordering, claims and revenue cycle aspects of the EHR. For clinicians I can see how this would be incredibly frustrating as you could spend hours perfecting each chart, but because the EHR doesn’t understand your area of medicine it can’t assist you in giving better care.
